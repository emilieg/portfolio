PROJECTS

WHAT'S THAT?

An app that translates scientific ingredient language into simple terms. Scan food, cleaning, and personal products by barcode and get detailed information about the ingredients.

This app was a group collaboration at the AngelHack Hackathon Seattle. We won 2nd place overall as well as the Code4Impact award.

PRODUCE EXCHANGE

A Craigslist like app for produce that helps neighbors trade their goods. Solo full stack project completed in 4 days. This application features user sign up, sign in and sign out functionality. I used MongoDB and Mongoose to persist post data and the Cloudinary API to upload pictures. The backend is built in Node.js and Express and the frontend is built with Angular.js.

WINO911

This is a full stack Node.js app that features user authentication and several external APIs. Wino911 provides you with directions to the closest wine bar. A local temperature display helps you decide whether you should drink red or white. And you can save your favorite bar to a favorites list. I used the user’s current coordinates to find a wine bar and route directions using the Google Places and Directions API.

CAPPSTONE

Cappstone is an app that helps you track all activities related to your job search. Check for upcoming appointments, add contacts, and add notes about meetings.

This app was build in Ruby on Rails as a group initiative with 3 other developers in 4 days. I was responsible for the design and front-end work, Google Calendar API and helped implement Google OAuth.

STUCK IN THE MUD

Stuck in the Mud is a dice game where two players compete for the highest score. They are only able to score points when their roll does not include any twos or fives. This is a front-end only project where I focused on design, interactivity and game loop script.
